What can frontier models do with this?

Ten tools expose your wiki as a knowledge graph, not just files:

  • list_domains · get_index — discover what's there
  • get_graph_overview — topology: hubs, clusters, orphans
  • get_tags — tag-based research across the wiki
  • search_wiki · search_cross_domain — ranked search with link-count enrichment
  • get_node · get_backlinks — pull a page with its full graph context
  • get_connected_nodes — multi-hop graph traversal
  • get_summary — trace knowledge back to source
To remove My Curator from Claude Desktop, delete the my-curator entry from claude_desktop_config.json and restart Claude Desktop.

Create your access token

This token is yours — it identifies your contributions. The admin never sees it. Other contributors never see it.

  1. Click the button below to open GitHub's token creation page. The name will be prefilled.
  2. Under Repository access, choose Only select repositories → pick the cohort repo.
  3. Under Permissions, click + Add permissions → add Contents → set to Read and write.
  4. Scroll down → click Generate token.
  5. Copy the token (starts with github_pat_) and paste it here.

Before you use Ask AI

When you click Ask AI, The Curator sends the following to your configured LLM provider (Google Gemini or Anthropic, whichever you set in Settings):

  • A short excerpt (~4 KB) of the wiki page that contains the broken link
  • A list of your wiki's page names (slugs only — not their contents)

The provider's privacy policy applies. To disable this feature, remove your API key in Settings. See docs/ai-health.md for details.
