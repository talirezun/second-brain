# Deployment
Tags: software

## Summary
The process of releasing and making software available for use.

## Key Aspects
- Encompasses the stages from development completion to live production.
- Key considerations include infrastructure, security, and scalability.
- Tools like [[git]] are crucial for managing deployment pipelines.
- Cloud platforms such as [[google-cloud]] and [[entities/digitalocean]] are commonly used for hosting.
- The Lumina AI case study successfully deployed a SaaS application using [[google-cloud]]'s Firebase hosting.
