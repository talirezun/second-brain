# Git
Type: tool
Tags: version-control, distributed, software-development, open-source

## Summary
A distributed version control system widely used for tracking changes in source code during software development.

## Key Facts
- Created by Linus Torvalds in 2005.
- Designed for speed, data integrity, and support for distributed, non-linear workflows.
- Allows developers to track every change made to their codebase, revert to previous versions, and collaborate effectively.
- Forms the backbone of platforms like GitHub, GitLab, and Bitbucket.
