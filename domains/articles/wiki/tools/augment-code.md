# Augment Code

## Summary

Augment Code is an AI coding tool that offers a unique "Context Engine" for deep semantic understanding of codebases, particularly effective for large and complex projects. Its ambitious "Intent" platform functions as an AI-led orchestration system, delegating tasks to specialized agents.

## Key Facts

*   Features a "Context Engine" that builds a semantic dependency graph of the entire codebase.
*   The "Intent" platform provides AI-led orchestration, with a coordinator agent managing specialist agents.
*   Can delegate work to a large number of agents (e.g., 177 agents observed).
*   Operates on a credit system, with costs varying significantly based on usage and the AI model used (e.g., Opus 4.6).
*   The pricing reflects the ambition of the build, making it a significant investment for intensive commercial projects.
