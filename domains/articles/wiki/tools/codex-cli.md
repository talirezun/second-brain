# Codex CLI

## Summary

Codex CLI is an open-source AI coding tool from OpenAI that emphasizes economics, transparency, and asynchronous delegation. It provides a cost-effective solution, particularly with its $200/month ChatGPT Pro plan, and is favored for its auditable code and efficient cloud execution model.

## Key Facts

*   Open-source tool with significant community traction (e.g., 67,000+ GitHub stars).
*   Utilizes OpenAI's models and offers a cost-effective pricing structure, with the $20 Plus plan being sufficient for lighter usage.
*   Employs asynchronous cloud execution, allowing tasks to run independently in a sandboxed environment for up to thirty minutes.
*   Known for strong code review capabilities, often catching errors missed by other tools.
*   Uses "agentic navigation" via shell commands and file reading; does not pre-index codebases.
*   The AGENTS.md file system provides cross-tool compatibility for project-specific instructions.
