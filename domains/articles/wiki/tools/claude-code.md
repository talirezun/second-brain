# Claude Code

## Summary

Claude Code, developed by Anthropic, is an AI coding agent that positions the human developer as the "conductor." It excels in its sophisticated four-layer memory architecture, dynamic "agentic search" for context, and its ability to leverage parallel sessions for enhanced productivity.

## Key Facts

*   Employs a "conductor" philosophy where the human orchestrates the AI orchestra.
*   Features a sophisticated four-layer memory architecture: Human-authored context (CLAUDE.md), Auto Memory, session memory, and Auto Dream (memory consolidation).
*   Uses "agentic search" to navigate codebases dynamically rather than pre-indexing.
*   Supports parallel sessions, allowing developers to manage multiple tasks concurrently.
*   Offers a cost-effective $200/month Max plan providing generous token allocation for Anthropic models.
*   Currently limited to using Anthropic models, with potential rate limiting as a constraint.
